# Contributing to Memory of Agents (MOA)

Thank you for your interest in contributing to MOA!

## Getting Started
1.  Fork the repository.
2.  Clone your fork: `git clone https://github.com/your-username/moa.git`
3.  Install dependencies (TBD).

## Making Changes
1.  Create a new branch: `git checkout -b feature/your-feature-name`
2.  Commit your changes: `git commit -m "feat: add new feature"`
3.  Push to your fork: `git push origin feature/your-feature-name`
4.  Open a Pull Request.

## Code Style
-   **Solidity**: Follow standard Solidity style guide.
-   **Python**: Follow PEP 8.
-   **TypeScript**: Use Prettier standard config.

## Reporting Issues
Please use the GitHub Issues tab to report bugs or suggest enhancements.
